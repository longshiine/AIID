# 복합인지 실종예측 모델 연구

## 1. 연구 목적 및 방향성
    a) 기존 목표

    원래 목표한 바

    목표: 실종위험 예측모델 개발

    - input: 영상에서 추출된 텍스트 기반 이벤트 데이터
      output: 실종위험 예측 및 위험 상황 인지 

    - LSTM 활용
    - 추가적으로 과거 실종 사건 데이터를 모델 훈련에 활용
    - 과거 사건 기준 + 현재 이벤트 기준 실종위험 예측을 얻게됨

    b) 변경 목표

    데이터의 한계에 의한 선회

    - 본래 목표는 영상에서 추출된 텍스트 기반 이벤트 데이터(xml파일)를 input으로 받는데, 이를 학습에 활용하기 위해선 이벤트 데이터 뿐만 아니라 해당 이벤트가 실제 실종 사건 발생으로 이어졌는지 그 여부에 대한 정보가 필요함
    - 위 언급된 데이터셋의 확보가 불가한 것으로 파악되어, 순수히 과거 실종 데이터만으로 실종예측 모델을 구축해야하는 상황에 다다름.
    - 따라서 과거 실종 데이터를 적절히 가공하여 이를 기반으로 다양한 예측모델을 제시해보고자 함

## 3. 예측 모델 1,2,3 - RF, DNN, LSTM 개요 및 성능 분석 ( 현재 성능(%): RF - 30 / DNN - 17 / LSTM - 91/5 )

    RF와 DNN은 특정 지역(동)에 대하여, 월, 요일, 기상 정보(기온(C), 강수량(mm), 풍속(m/s), 습도(%), 일조(hr), 일사(MJ/m2))를 input으로 넣었을 때 실종 발생 시간(시간 단위)을 output으로 주는 모델입니다.
    RF와 DNN 모델은 실종 사건이 발생한 케이스에 대한 데이터만으로 학습하기에, 어느 날이든 일단 실종 사건이 발생한다는 전제 조건 하에 언제(어느 시간에) 발생할지를 예측하는 것입니다.
    LSTM은 한 지역에 대해 실종이 발생하지 않은 날들에 대한 기상 데이터까지 수집하여 특정한 날에 실종이 발생할지 안할지 여부를 예측하는 모델입니다.

    a) Random Forest
    전처리 / 모델 / 성능
    b) Deep Neural Network
    전처리 / 모델 / 성능
    c) Long Short-Term Memory
    전처리 / 모델 / 성능
